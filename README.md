# Sweet Crumbs Bakery

A beautiful, warm, and welcoming website for **Sweet Crumbs Bakery** — an artisan bakery in Nairobi, Kenya.

## About the Project

This is a simple static HTML website for Sweet Crumbs Bakery. It showcases the bakery’s story, menu, weekly specials, customer reviews, and contact information using a cozy, traditional design with warm earth tones.

### Features
- Responsive table-based layout
- Hero section with call-to-action buttons
- About Us section with key statistics
- Full menu with prices and availability
- Weekly specials
- Image gallery placeholder
- Customer testimonials
- Contact form
- Location & map link
- Professional bakery branding

## Technologies Used
- HTML5
- CSS (inline styling with warm color palette)
- Table-based layout (classic approach)
